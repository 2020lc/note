```
//已提交
if(row?.status === 1) {
  if(row.canRetract && row.canDisuse) {
    显示,撤回、废弃按钮
  } else if(row.canRetract && !row.canDisuse) {
    显示,撤回按钮
  } else if(!row.canRetract && row.canDisuse) {
    显示,废弃按钮
  }
}
// 待提交
if(row?.status === 2) {
  if(row.canDisuse) {
    显示,提交、编辑、删除、废弃按钮
  } else {
    显示, 提交、编辑、删除按钮
  }
}
已提交状态
  只可能显示'撤回'或者'废弃'按钮
  根据后端'撤回''废弃'字段控制
待提交状态
  必显示'提交''编辑''删除'按钮
  '废弃'按钮,根据后端'废弃'字段控制
```
