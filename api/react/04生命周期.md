# 生命周期

**（旧）**
```js
  class Life extends React.Component {
    // 构造器
    constructor() {}
    // 组件将要挂载
    componentWillMount(){}
    // 子组件接收props, // 第一次不调用
    componentWillReceiveProps(props) {}
    // setState时调用，组件是否更新
    shouldComponentUpdate() {}
    // setState后调用，组件将要更新, 或者使用forceUpdate不通过shouldComponentUpdate，强制更新
    componentWillUpdate(){}
    // 组件渲染
    render() {}
    // setState后调用，组件更新完成
    componentDidUpdate() {}
    // 组件挂载完毕
    componentDidMount() {}
        // 组件将要卸载
    componentWillUnMount() {}
  }
```
