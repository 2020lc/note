# React高阶函数

```js
  class Login extends React.Component {
    state = {
      username: '',
      password: ''
    }
    // 保存表单数据到状态中
    saveForm = (dateType) => {
      return (e) => {
        this.setState({[dateType]: e.target.value})
      }
    }

    handleSubmit = (e) => {
      e.preventDefault();
    }

    render() {
      return (
        <form onSubmit={this.handleSubmit}>
          用户名：<input onChange={this.saveForm('username')} type="text" name="username" />
          密码：<input onChange={this.saveForm('password')} type="password" name="password" />
          <button>登录</button>
        </form>
      )
    }
  }

```

**函数的珂里化**
> 通过调用函数返回函数的方式，实现多次接收参数最后统一处理的函数编码形式
