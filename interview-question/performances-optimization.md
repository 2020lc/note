# 性能优化

## 弹框接口发送请求，响应时间过长
### 解决方案
1. 立即返回结果
   1. 前端发送请求后立即关闭弹框，并弹出提示等待信息
   2. 自动刷新一次页面数据，得到后端此次响应信息以及页面其他数据信息  
      | 响应状态       | 响应结果        |
      | -------------- | --------------- |
      |                | 未处理          |
      | pending        | 处理中          |
      | resolve/reject | 已处理/处理失败 |
   3. 根据响应状态设置按钮状态(loading或者禁用)
   4. 显示当前响应结果
